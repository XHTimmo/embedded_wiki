# 序章
在游玩本游戏之前，你需要安装游戏运行的环境，以及准备好一些基础知识
::: warning 系统需求
- 完成Keil的C51开发环境的安装
- 掌握hex文件烧录流程
- C语言基础
:::

# 开始之前
在正式开始C51单片机之旅之前，你需要阅读以下条款并认同
1. 这事C51的旅途，也是你的试炼，你只有最大限度的独立完成本试炼才能获得最大程度的提高，当然，你可以寻求他人的帮助，但是你需要表达对施救者的感谢，并且需要主动去掌握你求助的部分
2. C51的故事写的不够出彩，里面仍然有许多不合理的地方，如果你遇到问题，当你怀疑是故事本身的问题或者是设计的不合理的地方时，你可以直接在Github仓库提出你的Issue与PR，让我们一同完善C51的故事
3. 故事中的每一个主线任务的程序，你需要保存好代码实现并可以方便的进行展示，后续进行考核的时候会用到
